# Test Edilebilir Fizik

Bu bölüm B-HİG'in bilimsel olarak sınanabilir tarafını içerir.

Örnek testler:

- Spektral boyut: `d_s -> 3`
- Family sektörü: `Q_BHIG = 3`
- Foton: `m_gamma = 0`, `N_pol = 2`
- `N -> infinity` kararlılık testi
- Flavor-yük minimizasyonu
- Foton dispersiyonu sınırları

```text
v_gamma(E) = c [1 - eta_gamma (E/E_BHIG)^p]
d_s(E) = 3 - eta_s (E/E_BHIG)^p
```
